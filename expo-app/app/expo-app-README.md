# AD311 Expo App

This folder contains the Expo (React Native) portion of my AD311 “React Applications with Different Frameworks” assignment.

## Features
- Built with Expo and Expo Router
- Two screens: Home and About
- Navigation between screens using Expo Router

## How to Run
1. Install dependencies:
npm install
2. Start the Expo development server:
npm start
3. Press `w` to run the app in the browser.
