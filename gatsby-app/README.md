# AD311 Gatsby App

This folder contains the Gatsby portion of my AD311 “React Applications with Different Frameworks” assignment.

## Features
- Built with Gatsby
- Two pages: Home and About
- Client-side navigation using Gatsby's `<Link>` component

## How to Run
1. Install dependencies:
npm install
2. Start the development server:
npm run develop
3. Open the site:
- http://localhost:8000
