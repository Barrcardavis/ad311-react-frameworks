# AD311 Remix App

This folder contains the Remix portion of my AD311 “React Applications with Different Frameworks” assignment.

## Features
- Built with Remix 2.x
- Four routes:
  - `/` (Home)
  - `/about`
  - `/users`
  - `/users/:id`
- Uses loaders and actions where appropriate

## How to Run
1. Install dependencies:
npm install
2. Start the Remix development server:
npm run dev
3. Open the app:
http://localhost:3000
